# 白箱 SHIROBAKO
> アニメーション業界の今が、ここにある。  
> 动画业界的当下，就在此处！ ——广告

“想要进入动画行业，完成属于她们自己的作品！”这是某个高中动画同好会仅有的五名成员的梦想，《白箱》就是以这些女孩为中心，讲述她们进入业界后遇到的欢乐和挫折，进而描写动画人以及整个业界的奋斗的故事。
